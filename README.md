# Smart Plant Monitoring & Watering System

## Overview

An ESP32-based IoT-enabled plant care system that automatically waters plants based on soil moisture conditions while monitoring:

- Soil Moisture
- Temperature
- Humidity
- Water Tank Level

The system features OLED-based animated eyes, touch-controlled expressions, water-level alerts, Blynk cloud monitoring, custom PCB design, and CAD enclosure design.

---

## Features

1. Automatic irrigation

2. Soil moisture monitoring

3. Temperature and humidity monitoring

4. Water tank level detection

5. Low water level buzzer alert

6. OLED animated eyes

7. Touch-controlled facial expressions

8. Human detection using IR sensor

9. Blynk IoT remote monitoring

10. Custom PCB design

11. CAD enclosure design

---

## Hardware Used

- ESP32
- DHT11
- Soil Moisture Sensor
- HC-SR04 Ultrasonic Sensor
- OLED Display
- Relay Module
- Water Pump
- IR Sensor
- Touch Sensor
- Buzzer
- Logic Level Converter
