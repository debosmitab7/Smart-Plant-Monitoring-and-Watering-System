# Smart Plant Monitoring & Watering System

## Overview

An ESP32-based IoT-enabled plant care system that automatically waters plants based on soil moisture conditions while monitoring:

- Soil Moisture
- Temperature
- Humidity
- Water Tank Level

The system features OLED-based animated eyes, touch-controlled expressions, water-level alerts, Blynk cloud monitoring, custom PCB design, and CAD enclosure design.

---

## Features

✅ Automatic irrigation

✅ Soil moisture monitoring

✅ Temperature and humidity monitoring

✅ Water tank level detection

✅ Low water level buzzer alert

✅ OLED animated eyes

✅ Touch-controlled facial expressions

✅ Human detection using IR sensor

✅ Blynk IoT remote monitoring

✅ Custom PCB design

✅ CAD enclosure design

---

## Hardware Used

- ESP32

- Soil Moisture Sensor
- HC-SR04 Ultrasonic Sensor
- OLED Display
- Relay Module
- Water Pump
- IR Sensor
- Touch Sensor
- Buzzer
- Logic Level Converter
